## 10. Mermaid 流程图

### 流程图示例
```mermaid
graph TD
    A[开始] --> B{是否需要分析?}
    B -->|是| C[收集数据]
    B -->|否| D[结束]
    C --> E[LLM分析]
    E --> F[生成报告]
    F --> D
```

### 序列图示例
```mermaid
sequenceDiagram
    participant U as 用户
    participant F as 前端
    participant B as 后端
    participant L as LLM
    
    U->>F: 请求分析
    F->>B: 发送请求
    B->>L: 调用AI
    L-->>B: 返回结果
    B-->>F: 返回报告
    F-->>U: 显示报告
```

### 状态图示例
```mermaid
stateDiagram-v2
    [*] --> 待处理
    待处理 --> 处理中: 开始分析
    处理中 --> 已完成: 分析成功
    处理中 --> 失败: 分析失败
    已完成 --> [*]
    失败 --> [*]
```

### 饼图示例
```mermaid
pie title 信息源分布
    "科技" : 35
    "财经" : 25
    "社交媒体" : 20
    "新闻" : 15
    "其他" : 5
```
